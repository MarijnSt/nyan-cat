## NYAN CAT

Process

	- 	Create one cat with animation
		-	Add start and endposition in a class
		-	Transition property to move from one state to the other

Next step

	-	Second click resets everything
	-	Randomize one cat

Properties to randomize

	-	top (start en end)
	-	left (start en end)
	-	transform rotate (end)
	-	transition (element)
